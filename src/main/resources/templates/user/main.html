SMULET 이용안내

빠른 시작 (60초 요약)

1) 로그인/회원가입 — 우측 상단에서 진행합니다. 토큰 만료 시 자동 재발급 실패하면 재로그인하세요.
2) 마이페이지 — 시간표 만들기/불러오기, 과목 추가·삭제, 저장을 할 수 있어요.
3) 대표 시간표 지정 — 가장 최근 학기를 대표로 설정하면 메인에 표시됩니다.

핵심 기능 가이드

  • 로그인 노출 규칙 — 토큰이 있으면 “로그인/회원가입”은 숨김, “로그아웃/마이페이지”만 표시됩니다.
  • 마이페이지 이동 오류(401) — 자동 재발급 시도 후 실패하면 로그아웃 안내 및 재로그인 필요.
  • 시간표 입력 팁 — 같은 시간대 중복 추가 방지, 시작·종료 시간이 같으면 단일 시간 표시.
  • 대표 시간표 — 여러 학기 중 대표로 지정하면 메인 박스에 렌더링됩니다.

FAQ

  • 로그인 버튼이 안 보여요 — 이미 로그인 상태입니다. 로그아웃 후 다시 시도해보세요.
  • 마이페이지가 안 열려요 — 토큰 만료 가능성: 새로고침 후 재로그인.
  • 대표 시간표가 비어 있어요 — 마이페이지에서 시간표 생성/불러오기 후 “대표로 설정”을 눌러주세요.

정책 & 문의

개인정보처리방침 및 이용약관은 푸터 링크에서 확인하세요. 버그/제안은 깃허브(준비 중) 또는 담당자에게 문의해주세요.
